# Project Brief: ARMS Android Auto

## 프로젝트 목표
대한민국 주요 라디오 및 음악 스트리밍 방송을 안드로이드 오토 환경에서 편리하게 청취할 수 있는 앱 개발.

## 핵심 기능
1.  **라디오/음악 스트리밍 수신**: 대한민국 주요 라디오 방송 및 음악 스트리밍 방송 수신
2.  **차량 친화적 UI**: Android Auto의 디자인 가이드라인을 따르며, 차량 환경에서 조작하기 쉬운 인터페이스 제공
3.  **즐겨찾기**: 자주 듣는 방송을 즐겨찾기에 추가하여 빠르게 접근
4.  **현재 방송 정보**: 현재 청취 중인 방송의 상세 정보 (방송 편성 이름, 음악 제목, 방송국 데이터 등) 표시

## 제외 기능
1.  **커스텀 방송국 추가**: 사용자가 직접 방송국을 추가하는 기능은 제공하지 않음

## 로컬 테스트 환경
*   방송 수신 기능 테스트를 위한 별도의 CUI 또는 웹 기반 테스트 페이지 제공

## Harness Engineering
*   모든 개발 기록은 컨텍스트 파일로 기록하여 개발 과정의 투명성과 연속성을 확보

## 프로젝트 명
**ARMS Android Auto** (Automobile Radio & Music Streaming for Android Auto)
